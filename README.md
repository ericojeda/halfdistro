#Drupal 7 HCK2 Half Distro

##Installation
 
Clone into any directory  
`git clone https://github.com/ericojeda/halfdistro.git`
 
Make distro via Drush  
`drush make --prepare-install distro.make /path/to/your/dir`
 
##Included Modules  
  
* CKEditor
* CTools
* Nodequeue
* Strongarm
* Views

##Included Libraries
* Backbone
* Underscore
* Modernizr
